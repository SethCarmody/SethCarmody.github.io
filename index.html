Artifacts


My GIMM 110 final was an individual game that I created. The game is a Metroidvania in which the player can engage in combat with enemies as well as explore the levels. The project features two levels; a tutorial and one actual level.

* Note: All assests used were from the Unity store.

This project taught how to create games using the Unity Engine as well as giving me experince with coding, designing levels and planning.

This project was a redesign of a mobile app using Adobe XD. I choose to redesign the Hoyolab app to make it function as a Genshin Impact companion app.

The main page is organized between account tools and game tools. Account tools require you to log into an account to use while game tools are accesable for everyone. Users who are signed into an account can do various things that were not possible on the main app, such as adding and accepting friends and viewing your accounts wish history.

From this project, I learned basic UX and UI principles as well as how to use Adobe XD to make prototypes of mobile applications.
